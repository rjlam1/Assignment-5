Question-1:What are the different ways to select an element in to the Dom?

To select an element in the DOM using JavaScript, you can use methods like: getElementById (selects by unique ID), getElementsByClassName (selects by class name), getElementsByTagName (selects by HTML tag name), querySelector (selects using a CSS selector to find the first matching element), and querySelectorAll (selects all elements matching a CSS selector).

Question-2:What is the difference between innerHTML,innerTEXT,and textContent?

"innerHTML": Useful for manipulating the structure of an element by adding or modifying HTML content.
"innerText": Useful for extracting the text that a user would see on a page, ignoring styling and hidden elements.
"textContent": Useful for getting the raw text content of an element, including hidden text and whitespace, without interpreting HTML tags.

Question-3:What is event delegation in the DOM?

Event delegation refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated. It allows us to attach a single event listener for elements that exist now or in the future.

Question-3:What is event bubbling in the DOM?

In the DOM (Document Object Model), "event bubbling" refers to the behavior where an event triggered on a specific element travels up the DOM hierarchy, successively triggering the same event on its parent elements until it reaches the root of the document, essentially "bubbling" the event upwards through the DOM tree; this allows you to capture events on parent elements even if they were initiated on a child element within them.
